# mvc_janaina_atividade

## 📋 Descrição do Projeto
Projeto desenvolvido com arquitetura MVC (Model-View-Controller) para fins educacionais.

## 🛠️ Tecnologias Utilizadas
- **JavaScript** (69.3%) - Lógica da aplicação
- **EJS** (29.5%) - Template engine para views
- **HTML** (1.2%) - Estrutura das páginas

## 📁 Estrutura do Projeto
```
mvc_janaina_atividade/
├── pastinha/          # Pasta do projeto
├── package.json       # Dependências do projeto
├── package-lock.json  # Lock file das dependências
└── README.md         # Este arquivo
```

## 🚀 Como Começar

### Pré-requisitos
- Node.js instalado na sua máquina
- npm ou yarn

### Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/Dastical/mvc_janaina_atividade.git
   cd mvc_janaina_atividade
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

### Execução
Para iniciar a aplicação:
```bash
npm start
```

A aplicação estará disponível em `http://localhost:3000` (ou na porta configurada).

## 📚 Arquitetura MVC

Este projeto segue o padrão arquitetural MVC:

- **Model**: Responsável pelos dados e lógica de negócio
- **View**: Responsável pela apresentação (templates EJS)
- **Controller**: Responsável por gerenciar requisições e respostas
